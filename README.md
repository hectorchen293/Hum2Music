🎵 哼唱编曲助手 Pro v2.1
把你的哼唱变成完整编曲的桌面工作站
支持多轨录音、旋律分析、智能歌词识别、乐器叠加、鼓轨生成、混音导出，让灵感永不丢失。

✨ 核心功能
🎤 多轨录音
一键录音：点击主录音键立即录制人声（默认从 0 秒开始）。

重唱叠加：支持新建音频轨并指定起始时间，轻松录制和声、叠唱。

每条音轨独立控制：音量、静音、独奏、删除。

📊 旋律编辑
自动分析：从任意音频轨中提取音高，生成 MIDI 音符表格。

手动编辑：可添加、删除、双击修改音符的时间/音高，支持半音调节。

旋律表格实时显示，是乐器生成的依据。

📝 智能歌词识别
内置 Whisper 模型（默认 small），支持粤语/中文等语言。

自动生成带时间戳的歌词，可手动编辑，用于后续字幕导出。

🎹 乐器与鼓轨
多种乐器：钢琴、吉他、小提琴、长笛、小号，可移调（±12 半音）。

鼓轨生成：选择入鼓小节，自动生成 kick/snare/hihat 基础节奏。

所有乐器基于当前旋律表格实时合成（FluidSynth + SoundFont）。

🔊 多轨混音
同时播放所有未静音轨道，支持独奏模式。

每条轨道独立调节偏移时间和音量，完美对齐重唱。

实时试听，停止控制。

🎬 导出
音频导出：WAV / MP3 格式。

视频导出：生成带歌词字幕的 MP4 视频（黑底 + 同步歌词），适合分享。

🛠 自动依赖管理
首次运行自动检测并安装缺失 Python 库。

自动下载高质量 SoundFont（Timbres of Heaven，约 150MB），也可手动放置。

需提前安装 FluidSynth 和 FFmpeg（用于音视频合成）。

📥 安装与运行
系统要求
Windows 10/11（macOS/Linux 理论上可运行，部分路径需调整）

Python 3.8+（推荐 3.10 以上）

必须安装的外部工具：

FluidSynth（加入 PATH）

FFmpeg（加入 PATH）

运行步骤
安装 Python 并下载本程序（编辑.py）。

安装 FluidSynth 与 FFmpeg，确保命令行可运行 fluidsynth 和 ffmpeg。

双击 编辑.py 或执行 python 编辑.py。

程序会自动安装缺失的 Python 库，并尝试下载 SoundFont。

若 SoundFont 下载失败，可手动下载 Timbres_Of_Heaven_GM.sf2 放在程序同目录。

📖 使用指南
步骤	操作
1. 录制人声	点击 “● 开始录音” 直接录音；或点击 “新建重唱轨” 输入起始时间后录制。
2. 分析旋律	点击 “分析选中旋律”，系统从第一个音频轨提取音高并显示在旋律编辑器。
3. 编辑旋律	在旋律表格中双击修改音符或时间；用下方按钮添加/删除音符。
4. 识别歌词	点击 “识别歌词”，模型自动识别并显示在歌词框（支持粤语）。
5. 添加乐器	点击 “添加乐器轨”，选择乐器和移调，自动根据旋律生成伴奏。
6. 添加鼓	点击 “添加鼓轨”，选择入鼓小节，自动生成节奏。
7. 混音调整	在音轨列表中调节每条轨道的音量、偏移、静音/独奏。
8. 试听	点击 “▶ 播放” 实时监听混音效果；点击 “⏹ 停止” 中断。
9. 导出	“导出音频” 保存 WAV/MP3；“导出 MP4” 生成带歌词字幕的视频。
⚠️ 常见问题
Q: 提示 FluidSynth 错误或合成失败
A: 检查 FluidSynth 是否安装并加入 PATH，SoundFont 文件是否正确（Timbres_Of_Heaven_GM.sf2 放在程序目录）。若超时，可在代码中将 timeout 调大。

Q: 歌词识别很慢或下载模型卡住
A: Whisper small 模型首次使用需下载约 460MB，建议使用手机热点或提前用镜像下载后放入 %USERPROFILE%\.cache\whisper。

Q: 听不到人声？
A: 确保人声轨道未静音，且音量不为 0。若混音后声音小，可调高该轨音量滑块。

Q: 如何录制粤语歌？
A: 直接录制后点击“识别歌词”，程序已设置为中文识别（含粤语）。如需更高准确度，可修改代码中 language 参数为 "yue"。

Q: 导出 MP4 无字幕？
A: 需要安装 FFmpeg 且加入 PATH。若无 FFmpeg，程序会降级输出无字幕视频。

视频合成：moviepy + FFmpeg

🚀 高级玩法
和声重唱：录制多轨人声，设置不同偏移时间，调节音量创造立体声效果。

乐器合奏：添加钢琴、吉他、小提琴轨，分别移调八度，构建丰富伴奏。

鼓加花：可手动编辑鼓轨的 MIDI 生成函数，自定义节奏型。
